## 移动光标
* ctrl + b 后移一页
* ctrl + f 前移一页
* ctrl + u 后移半页
* ctrl + d 前移半页
* g+g 回到开头
* G 回到最后
* $ 移到行尾
* ^ 移到行首
* #l 移到该行的第几个位子，如5l移到第5个位子

## 删除文字
* x 删除光标后面一个文字
* #x 删除#个文字
* X 删除光标前面一个文字
* #X 删除光标前面#个文字
* d+d 删除一行

## 撤销
* u 取消上次更改

## 复制
* p 将缓冲区字复制到光标所在位置
* y+y 将该行复制到缓冲区
* #yy 将#行复制到缓冲区
* y+w 复制光标所在文字
* #yw 复制#个文字

## 其他
* : set number 设置行号
* : # 跳转到#行
* / + 关键词 向后寻找关键词，按n跳转
* ? + 关键词 向前寻找关键词，按n跳转
